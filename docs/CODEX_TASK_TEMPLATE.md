## 0. 前提

- YUITO の `CODING_GUIDELINES.md` と `CODEX_RULES.md` を必ず遵守すること
- views.py に業務ロジックは禁止
- fat model 禁止

- 既存 API 仕様・レスポンス形式・例外からHTTPへのマッピング・テスト期待を壊さないことを最優先とする
- 不明点があれば勝手に実装せず質問する
