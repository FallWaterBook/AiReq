# AiReq Mobile Prompt Template

以下の形式で、AiReqに投入するための実装依頼プロンプトを作成してください。

## 前提

- このプロンプトは、別のAI開発支援ツールに入力します
- そのツールはDjango/Pythonコードを修正します
- 出力形式（JSONなど）はツール側で制御されます
- あなたは「実装依頼内容のみ」を作成してください

## 必須構成

以下の構成で必ず作成してください。

### 目的
何を実現したいか簡潔に書く

### AS-IS（現状）
現在どうなっているかを書く

### TO-BE（理想）
どのように変更したいか具体的に書く

### WHY（理由）
なぜその変更が必要かを書く

### 対象ファイル（分かる場合）
例:
- jobs/views.py
- templates/jobs/form.html

### 制約
- 最小変更
- 既存機能を壊さない
- 大きなリファクタリング禁止
- Django構成を守る
- セキュリティ制約を維持
- 不明点は推測せず、安全な範囲で実装する
